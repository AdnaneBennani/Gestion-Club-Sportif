# Présentation PFE — Application Web de Gestion d'un Club Sportif

Cette présentation est une version **web responsive** (20+ slides) prête pour soutenance.

## Lancer la présentation

Option 1 (simple):
- Ouvrir `index.html` directement dans le navigateur.

Option 2 (recommandée):
- Lancer un serveur local (ex: VS Code Live Server).

## Navigation
- Flèches clavier `←` `→`
- Boutons `Précédent` / `Suivant`
- Sommaire cliquable

## Personnalisation rapide
Modifier `data/slides.json` pour adapter:
- contenu textuel
- titres
- items de timeline/checklist
- difficultés/solutions
- perspectives

## Captures d’écran & diagrammes
Déposer vos images dans `assets/` puis mettre à jour les chemins dans `data/slides.json`.

Sections prévues:
- Connexion
- Tableau de bord
- Membres
- Équipes
- Entraîneurs
- Entraînements
- Paiements
- Profil
- UML (use case, classes, séquence, activité)
- Base de données (MCD, relationnel)
- Architecture générale

## Export PDF
Depuis le navigateur:
1. `Ctrl+P`
2. Destination: **Enregistrer au format PDF**
3. Mise en page: **Paysage**
4. Marges: **Aucune** (ou minimales)
5. Activer arrière-plans
